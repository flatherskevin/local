# Week 4: Master

Week 4 is about owning the environment.

You survived, you navigated, you found flow. Now you make it yours. This week you stop following instructions and start customizing. You learn to use AI as a tool beside your editor, not a replacement for thinking. You tune your config files. You push into advanced Vim patterns. You measure your speed and find the remaining friction. By Friday this is not a course setup anymore. It is your setup.

## Success Criteria

By the end of this week you should be:

- **Using Claude Code deliberately** -- knowing when AI helps and when it gets in the way.
- **Customizing your config** -- adding aliases, keymaps, and bindings that fit your workflow.
- **Using advanced Vim patterns** -- WhichKey menus, text objects, visual block mode, and macros.
- **Working at speed** -- opening, navigating, editing, formatting, and committing without conscious thought.

## How to Use This Week

This is the last structured week. After this you practice on your own. Treat each day seriously. The exercises are less about learning new keys and more about internalizing what you already know. If something still feels slow, that is the thing to focus on.
