# Chapter 7: Run, Week 3

Week 3 is about feeling fast rather than merely functional.

## Your Goal This Week

By the end of week 3, the setup should feel like your normal working
environment.

You should no longer be thinking constantly about the tools themselves. You
should be thinking about the code and using the tools as support.

## Stronger Habits

- keep one repo per tmux session
- use tmux windows to separate tasks
- use search and LSP navigation more than the tree
- use formatting and diagnostics as defaults

## Small Motion Upgrades Worth Learning

- `ciw`
- `di(`
- `%`
- `*`
- `gg`
- `G`

You do not need a huge Vim vocabulary. A few high-value motions go a long way.

## What To Avoid

- do not chase advanced Vim tricks just because they look impressive
- do not keep adding panes instead of organizing work into windows
- do not use AI to compensate for weak navigation habits

The target is calm, repeatable speed.
