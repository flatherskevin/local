# Week 2: Navigate

Week 2 is about turning the setup into a repeatable work system.

You survived Week 1. You can open files, make edits, and close things without panic. Now you stop fumbling and start moving with intent. The goal this week is deliberate navigation: knowing where you are, getting where you need to be, and never reaching for the mouse.

## Success Criteria

By the end of this week you should be:

- **Using tmux windows and panes deliberately** -- splitting, resizing, and switching without hesitation.
- **Relying on LSP navigation** -- jumping to definitions, finding references, and reading hover docs instead of manually scanning code.
- **Using search more than visual scanning** -- reaching for F5/F6/F7 before scrolling through a file tree.

## How to Use This Week

One day at a time. Do the exercises in order. Each day builds on the previous one. If a day feels easy, good -- that means the muscle memory is forming. If a day feels hard, repeat it tomorrow before moving on.
