# Chapter 9: Kitty Workflow

Kitty is intentionally simple in this setup.

## Role Of Kitty In A Human Workflow

Kitty should help you start work and separate major contexts. It should not be
where you try to manage every detail of the day.

## What Kitty Owns

- OS windows
- top-level tabs
- quick terminal splits

## Useful Keys

- `kitty_mod+Enter`: new window
- `kitty_mod+t`: new tab
- `kitty_mod+d`: vertical split
- `kitty_mod+Shift+d`: horizontal split
- `F6`: new tab
- `F7`: vertical split
- `F9`: attach to the `main` tmux session
- `F10`: run `dev`

## Practical Rule

Use kitty to start or separate major workstreams.

Use tmux for the persistent work inside each workstream.

If tmux is your desk, kitty is the room the desk sits in.
