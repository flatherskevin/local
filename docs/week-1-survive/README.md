# Week 1: Survive

Week 1 is about surviving comfortably. You will not become fast this week. You will become functional. By Friday you should be able to open any repo, find files, search text, make edits, and run commands beside your editor without reaching for a mouse or switching to a different app.

## Success Criteria

By the end of this week you can:

- Open any project into a tmux session with `dev`
- Find files by name and search text across a project
- Make basic edits in Neovim (insert, save, quit, undo)
- Format code and review changes with LazyGit
- Run shell commands in a pane beside your editor

## How to Use These Guides

Each day introduces a small set of new concepts, then gives you concrete exercises to practice them. Do not skip the exercises. Muscle memory only comes from repetition.

Read the "New Concepts" section first. Then work through each exercise step by step. Finish by checking off every item in the "Success Check" list. If you cannot check an item, repeat the relevant exercise.

One day per sitting. Do not rush ahead. Five short sessions beat one long cram.
